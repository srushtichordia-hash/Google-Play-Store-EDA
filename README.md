# 📱 Google Play Store Data Analysis & EDA

## 📌 Project Overview

This project focuses on cleaning, preprocessing, and performing Exploratory Data Analysis (EDA) on the Google Play Store dataset.

The objective is to uncover meaningful insights about app ratings, installs, pricing, categories, and user engagement while handling real-world data quality issues.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🧹 Data Cleaning Performed

* Removed duplicate records
* Handled missing values
* Cleaned Reviews column
* Converted Size column into numeric format
* Converted Installs into numerical values
* Converted Price into float values
* Processed Android Version column
* Extracted date features from Last Updated
* Corrected inconsistent data formats

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

* Category Distribution Analysis
* Rating Distribution Analysis
* Install Distribution Analysis
* Price Analysis
* Content Rating Analysis
* Free vs Paid Apps Analysis
* Most Popular App Categories
* Correlation Analysis

---

## 🔍 Key Insights

* Most applications on the Play Store are free.
* Family and Game categories contain the highest number of apps.
* Ratings are concentrated between 4.0 and 4.5.
* Paid applications represent only a small fraction of total apps.
* Apps with higher install counts generally receive more reviews.
